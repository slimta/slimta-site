slimta.smtp¶

Root package for slimta SMTP client and server libraries.

exception slimta.smtp.SmtpError¶

Bases: slimta.core.SlimtaError

Base exception for all custom SMTP exceptions.

exception slimta.smtp.ConnectionLost¶

Bases: slimta.smtp.SmtpError

Thrown when the socket is closed prematurely.

exception slimta.smtp.MessageTooBig¶

Bases: slimta.smtp.SmtpError

Thrown when a message exceeds the maximum size given by the SMTP SIZE extension, if supported.

exception slimta.smtp.BadReply(data)¶

Bases: slimta.smtp.SmtpError

Thrown when an SMTP server replies with a syntax-invalid code.

Parameters:data – The data that was expected to start with an SMTP code, made available in the data attribute.
