slimta.queue.proxy¶

Package providing an alternative to the standard Queue class, such that messages are not written to storage but instead immediately relayed. An Edge service will have to wait for a message to finish relaying before its reply can be issued.

class slimta.queue.proxy.ProxyQueue(relay)¶

Bases: object

Class implementing the same interface as Queue, but proxies a message to a Relay service instead of storing and trying/retrying delivery.

Parameters:relay – Relay object used to attempt message deliveries.
