slimta.logging.subprocess¶

Utilities to make logging consistent and easy for any any subprocess operations.

class slimta.logging.subprocess.SubprocessLogger(log)¶

Bases: object

Provides a limited set of log methods that slimta packages may use. This prevents free-form logs from mixing in with standard, machine- parseable logs.

Parameters:log – logging.Logger object to log through.
